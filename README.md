Ferrugo is a JVM implementation written in Rust.

*Just for fun*

*JVM in Rust. Sounds great, isn't it?*

# Try it

1. Build

```
$ cargo run --release
```

2. Try an example

```
$ cargo run examples/Hello.class
```
